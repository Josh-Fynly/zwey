##Zwey## - Social Hub for Upcoming Artists
A social profile app where upcoming artists link all their music platforms in one place, share tracks, connect with each other, and distribute to Spotify/Apple Music in 1 click.

##Quick Links##
Schema: See SCHEMA.md
Timeline: See WEEKS.md

Tech Stack
Frontend: Next.js + React
Backend: Firebase (Auth + Firestore)
Hosting: Vercel
Styling: Tailwind CSS

##Getting Started
#*Prerequisites*#
Node.js 18+
Firebase account
Vercel account (free tier)

##Setup
Install dependencies:
npm install

##Configure environment:
Copy .env.example to .env.local
Add your Firebase credentials

##Run locally:
npm run dev
Open http://localhost:3000

##Deploy:
vercel deploy
